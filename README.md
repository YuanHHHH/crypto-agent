# Crypto Agent

AI-powered crypto trading analysis agent

## 功能计划

- 实时查询加密货币价格（CoinGecko API）
- AI 生成行情分析报告（LLM API）
- 多交易所价差监控
- 技术指标分析（RSI/MACD）
- 链上数据分析

## 技术栈

- Python
- FastAPI
- LLM API（Claude / OpenAI）
- CoinGecko API

## 安装

git clone https://github.com/你的用户名/crypto-agent.git
cd crypto-agent
pip install -r requirements.txt
cp .env.example .env  # 填入你的 API key

## 运行

python -m src.main
